Användarinformation

Denna statistikapplikation visar information om hur många meddelanden som förmedlas av Region Stockholms regionala tjänsteplattformar. Det finns två grundlägen:

  1. Antal anrop fördelat på respektive tjänstekonsument, tjänsteproducent, tjänstekontrakt och logisk adress (mottagning).
  2. Antal anrop per dag för ett specifikt urval. Detta läge nås genom att välja "Visa utveckling över tid".

Historik är tillgängligt tillbaka till 2017-07-30. Man väljer tidsintervall via vallistorna uppe till vänster. Vid uppstart visas senaste hela kalendermånad.

Filtrering

Användaren kan välja (klicka på) valfritt element i tabellen eller motsvarande tårtbit i pajdiagrammet. Det medför att filtrering sker på detta element. Genom flera val kan man "borra sig ner" till detaljerad information om anropsvolymer.

Via rutan "Visa" kan ett förval väljas. Det definierar ett specifikt urval av element som motsvarar det en specifik tjänst använder. Väljs exempelvis "Tidbokningar" visas anrop över tjänstekontrakt som hanterar tidbokningar. Dessutom anpassas rubrikerna till aktuellt förval.

Övriga funktioner

  • Knappen Visa allt går från att visa ett element till fyra.
  • Knappen Se urval i hippo växlar till hippo och tar med sig (delar av) aktuellt urval. Denna funktion kommer att förbättras i kommande versioner.
  • Knappen Exportera exorterar ut aktuellt urval i en CSV-fil. Den kan exempelvis öppnas i Excel.
  • Valet Tekniska termer växlar till de termer som används inom RIV-TA för att beskriva integrationer.

Anslutning av ytterligare tjänsteplattform

En förutsättning för denna statistik kan presenteras i denna vy är att aktuell tjänsteplattforms tjänsteadresseringskatalog (TAK) finns tillgänglig via Ineras TAK-api, se http://api.ntjp.se/coop/doc/index.html. Dessutom måste anropsdata tillhandahållas via en JSON-fil. Kontakta Regions Stockholms Förvaltningsobjekt Informationsinfrastruktur för mera information.
